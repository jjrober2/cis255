TWENTY USEFUL CSS PROPERTIES

Introduction

This file is to demonstrate basic competencies of
twenty useful CSS properties. A variety of these properties
are equally important, however I will be discussing 5 properties
and their respective function.

The Border-Style Property

This extension of the border property allows a developer to choose from
various attractive styles.
These styles can offer basic, to a more
formal border.

Solid

Double

Outset

The Background-Color Property

This property defines the color that
will be used to surround text present on the page.
According to w3schools.com, "All modern browsers support
140 color names." Here are just a few examples:


Dark Cyan

Dodger Blue

Midnight Blue

The Image Property

This property provides various controls
for images used on a website. A developer can
adjust the size, alignment, and background position
of an image.

The Overflow Property

Overflow will let the browser
know if any content in a container is bigger than the
container itself. It can have one of two states:

Hidden:

The hidden overflow property will simply hide
any content that does not fit inside the container.

Scroll: (please use scroll wheel to view)

The scroll property will provide a scrollbar
so one can scroll to see any hidden content.

The Float Property

The float property allows you to
display content, in normal flow, as far
left or right as deemed appropriate.
